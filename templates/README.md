# Templates

Reference templates for authoring content in this repository. **Never edit these originals** — copy their content into the target file, then fill in the placeholders.

| Template | Destination in repo | Purpose |
|---|---|---|
| `chapter-notes-template.md` | `[chapter]/notes.md` | Standard chapter notes — the core learning content |
| `thought-leadership-template.md` | `[chapter]/thought-leadership.md` | Blog/LinkedIn/talk angle for the chapter topic |
| `interview-prep-template.md` | `[chapter]/interview-prep.md` | Interview questions and strong-answer frames |
| `lab-template.md` | `[chapter]/LAB-XX-name.md` | Hands-on lab (global LAB numbering) |
| `module-index-template.md` | `[module]/README.md` | Module-level index (created on request) |
| `section-index-template.md` | `[section]/README.md` | Section-level index (exists at scaffold) |
| `capstone-template.md` | `capstone/project-brief.md` | Capstone project brief |
| `authoring-guidelines.md` | — (reference only) | Quality rubric and depth rules for all authoring |

See `AGENTS.md` in the repo root for the full content depth rules and conventions.
